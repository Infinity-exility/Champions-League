# Champions League

Projeto desenvolvido para praticar lógica de programação, manipulação de dados e estruturação de aplicações utilizando JavaScript.

O sistema simula funcionalidades relacionadas à uma Champions League, permitindo trabalhar conceitos como:

- Arrays
- Estruturas de repetição
- Condicionais
- Filtragem de dados
- Soma e contagem de informações
- Organização de partidas e resultados

## Tecnologias utilizadas

- JavaScript
- Node.js

## Objetivo

O objetivo deste projeto é fortalecer conhecimentos em programação e resolução de problemas através de desafios práticos inspirados em cenários esportivos.
